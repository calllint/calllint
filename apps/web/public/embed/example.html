Reference partner embed

A directory or partner shows CallLint's observed-at-digest verdict with one tag — without running CallLint. The widget reads the public, read-only Partner API (calllint.partner-api.v0). It is heuristic decision support, not a certification or guarantee of safety; UNKNOWN is never SAFE.

The inner <a> is the no-JavaScript fallback: if scripts are blocked, the plain link to the Trust Page is shown unchanged.
