# Agent examples

Copy-and-quote examples for coding agents and tool builders. Each input config
produces a real CallLint verdict:

| Input | Verdict | Finding | Report |
| --- | --- | --- | --- |
| `risky-mcp.json` | BLOCK | `files.broad-path` | `calllint-block-report.json` |
| `review-mcp.json` | REVIEW | `supply.unpinned-package` | `calllint-review-report.json` |
| `unknown-mcp.json` | UNKNOWN | `supply.unknown-remote` | `calllint-unknown-report.json` |

See `agent-summary-examples.md` for input → report → recommended wording, and
`/agent-instructions.md` on the site for the recommendation rules. The reports
are generated by the CLI (not hand-written); regenerate them with the commands
in `agent-summary-examples.md`.
