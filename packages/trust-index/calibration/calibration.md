## How to record a sign-off (human only)

Append to `packages/trust-index/calibration/review-store.json` under the artifact's
`artifactDigest`, then run `pnpm audit:calibration:write` and commit:

```json
{
  "schema": "calllint.calibration-review.v0",
  "signoffs": {
    "sha256:<artifactDigest>": [
      { "reviewer": "<handle>", "decision": "confirmed", "at": "<ISO-8601 UTC>", "note": "<why>" }
    ]
  }
}
```

`decision` is the reviewer's own judgment of whether the negative conclusion is a true
positive; it is NEVER a re-scored verdict and can never move the page's verdict.
