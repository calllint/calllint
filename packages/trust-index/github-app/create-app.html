Create the CallLint Trust GitHub App

This posts the least-privilege manifest (github-app/app-manifest.json) to GitHub. GitHub creates the App and returns its private key + webhook secret once — paste those into the ingestion repo's Actions secrets (see github-app/SETUP.md).

Leave the org blank to create it under your personal account, or enter an organization login to create it there (you must be an org owner).

The manifest requests only Repository metadata: read-only and subscribes to installation + installation_repositories. No code, contents, write, or PII scopes.
