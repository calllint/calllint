# ADR 0001: v0.1 Scope

Status: Accepted

## Decision

MCPGuard v0.1 is a CLI-first, JSON-native security verdict engine for MCP server / agent
skill configurations. It ships: config parsing, runtime-binding resolution, deterministic
static detectors, a risk engine, policy-as-code, fingerprints, a core pipeline, multiple
renderers, and a `mcpguard` CLI — backed by golden fixtures and tests.

## Out of scope

Gateway, payment, marketplace, user accounts, SaaS dashboard, large-scale crawler,
host execution of unknown servers, real-secret handling, destructive tool calls.

## Reason

The minimal value loop is: paste a config → get an evidence-backed verdict faster and
clearer than reading the config yourself. Everything else is downstream of that loop
earning trust.
